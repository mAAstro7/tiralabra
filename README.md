# tiralabra

Aiheena Kivi-Paperi-Sakset Tekoäly.

Pelissä on tekstipohjainen käyttöliittymä jota pelaaja voi käyttää yksinkertaisilla kommennoilla. Aluksi nimetään oma pelaaja sekä botti haluamilla nimillä.
Tämän jälkeen peli etenee antamalla seuraava siirto (k,p,s).
Kierroksen jälkeen näytetään pelaajalle kumpi voitti, mikä oli tekoälyn siirto ja mikä on tilanne pelissä. Pelissä käytettävät komennot ovat k,p,s ja stop
